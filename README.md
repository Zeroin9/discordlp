# Discord Points Bot

Discord-бот для управления системой очков/рейтинга пользователей на сервере Discord. Бот автоматически начисляет баллы лояльности (LP) пользователям, находящимся в голосовых каналах.

## Технологии

- **Язык**: Java 21
- **Фреймворк**: Spring Boot 4.1.0 (WebMVC, Data JPA, Thymeleaf)
- **Discord API**: JDA 6.4.2
- **База данных**: PostgreSQL
- **Миграции БД**: Flyway
- **Сборка**: Gradle

## Функциональность

### Автоматическое начисление баллов
Бот автоматически начисляет баллы лояльности (LP) каждые 5 минут пользователям, находящимся в голосовых каналах:

- **Заглушенный участник**: 0 LP
- **Обычный участник**: 100 LP
- **Стример** (ведет активный стрим): 200 LP
- **Зритель** (в канале со стримером, но не стримит): 150 LP

### Команды
- `/lp` - просмотр вашего баланса
- `/lpkick` - отключить участника за 10 000 LP
- `/lpuser` - просмотр баланса участника (только для администраторов)
- `/lpadd` - начисление баллов участнику (только для администраторов)
- `/lpremove` - списание баллов у участника (только для администраторов)

## Установка и запуск

### Требования
- JDK 21
- PostgreSQL

### Конфигурация
Создайте файл `.env` на основе `.env.example` и заполните необходимые параметры:
- `DISCORD_BOT_TOKEN` - токен бота Discord
- Параметры подключения к базе данных PostgreSQL

### Сборка и запуск

```bash
# Проверка компиляции
.\gradlew.bat compileJava

# Сборка проекта
.\gradlew.bat build

# Запуск приложения
.\gradlew.bat bootRun
```
